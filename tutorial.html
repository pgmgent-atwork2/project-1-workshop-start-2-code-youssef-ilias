Hoe bouwen we zoiets!?

zo een website als dit bouwen met

  • HTML
  • CSS
  • JavaScript

Stap 1 - mappen structuur

om code te schrijven hebben we een code-editor nodig, we zullen Visual code studio gebruiken.

  • 1.1 download de startcode en plaats deze op je bureablad
  • 1.2 pak de .zip bestand uit en open het in Visual code studio met file > open folder
  • 1.3 ga naar de tablad Extenions, zoek en installeer de extensie Live server. na het installeer open je de index.html bestand en klik op de knop "Go Live" rechts onderaan de pagina.

stap 2 - Time to code

laten we wat code schrijven, geen schrik ik zal je tonen wat je moet en kan doen

  • Stap 2.1 - Kopieer en plak de html code in de index.html bestand voor het structuur van de website
  • stap 2.2 - kopieer en plak de CSS code in breaker.css bestand om wat stijl te geven aan de website

    je kan eventueel de achtergrond kleur van de webpagina en de canvas verander door background-color aan te passen naar jouw gewenste kleur
